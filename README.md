# 📰 FAKE NEWS DETECTION AI MODEL

A Machine Learning-based web application that detects whether a news article is **Fake 🛑** or **Real ✅** using Natural Language Processing (NLP).

---

## 🚀 Features

* 🔍 Detect fake vs real news instantly
* 🧠 Trained using Machine Learning (Logistic Regression)
* 📊 Text preprocessing & vectorization (TF-IDF)
* 🌐 Interactive web app using Streamlit
* ⚡ Fast and simple UI

---

## 🛠️ Tech Stack

* **Python**
* **Scikit-learn**
* **Pandas & NumPy**
* **Streamlit**
* **TF-IDF Vectorizer**

---

```
FAKE_NEWS_DETECTION/
```

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ankitkr756208-hash/FAKE_NEWS_DETECTION_AI_MODEL.git
cd FAKE_NEWS_DETECTION_AI_MODEL
```

### 2️⃣ Create virtual environment (optional but recommended)

```bash
python -m venv venv
venv\Scripts\activate   # For Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

Then open:

```
http://localhost:8501
```

---

## 🧠 Model Details

* Algorithm: **Logistic Regression**
* Vectorization: **TF-IDF**
* Dataset: Fake & Real News Dataset
* Accuracy: ~**90%+** (depends on training)
